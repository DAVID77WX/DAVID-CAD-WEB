# DAVID CAD Simulator

![Banner](https://via.placeholder.com/800x200.png?text=DAVID+CAD+Simulator)

A modern, premium web‑based CAD simulation tool built with **Vite**, **TypeScript**, and **Tailwind CSS**. The interface is styled to resemble Autodesk’s sleek dark‑theme aesthetic, providing a responsive and dynamic user experience.

---

## 📦 Features

- **Dynamic UI** – glass-morphism cards, smooth gradients, and micro‑animations.
- **Responsive Layout** – works on desktop, tablet, and mobile screens.
- **Hot‑Reload Development** – powered by Vite for instant feedback.
- **Typed Architecture** – TypeScript ensures maintainable code.
- **Tailwind Powered** – utility‑first styling with a curated color palette.

## 🚀 Getting Started

### Prerequisites

- **Node.js** (v18 or later)
- **Git** (configured with your user name/email – already set in this repo)

### Installation

```bash
# Clone the repository (if you haven't already)
git clone <repo‑url>
cd projeto

# Install dependencies
npm install
```

### Development

Run the development server with hot‑reloading:

```bash
npm run dev
```

Open your browser at `http://localhost:5173` (or the URL shown in the terminal).

### Build for Production

```bash
npm run build
```

The optimized static files will be placed in the `dist/` folder, ready for deployment.
